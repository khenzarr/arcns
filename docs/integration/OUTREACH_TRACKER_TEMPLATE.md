## Status Taxonomy

| Status | Meaning |
|--------|---------|
| `target` | Identified as a priority partner — not yet contacted |
| `contacted` | First outreach sent — no response yet |
| `responded` | Partner replied — conversation open |
| `package-sent` | Full integration package delivered |
| `in-review` | Partner team is reviewing the package |
| `call-scheduled` | Intro or technical call booked |
| `integrating` | Partner has confirmed intent and is building |
| `stalled` | No response after 2+ follow-ups, or conversation went cold |
| `closed` | Integration live, or partner declined |
